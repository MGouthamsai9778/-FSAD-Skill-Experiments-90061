# Skill 12: Server-Side Rendering with Thymeleaf

This skill demonstrates server-side rendering using Spring Boot and Thymeleaf.

## Features
- Dynamic HTML generation
- Template processing
- Static resource serving

## Technologies Used
- Spring Boot 3.2.0
- Thymeleaf
- Java 17

## Running the Application
```bash
mvn spring-boot:run
```

The application will start on port 8082.

Visit http://localhost:8082 to see the rendered page.
